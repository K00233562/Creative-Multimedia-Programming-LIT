Web Programming

Overview

Web programming refers to the writing, markup and coding involved in Web development, which includes Web content, Web client and server scripting and network security. The most common languages used for Web programming are XML, HTML, JavaScript, Perl 5 and PHP. Web programming is different from just programming, which requires interdisciplinary knowledge on the application area, client and server scripting, and database technology.(Source) This is a module that is dedicated to Javascript. It covers all the basics and shows how to implement Javascript on websites for interactivity and validation. What is JavaScript? An object-oriented computer programming language commonly used to create interactive effects within web browsers

Continuous Assessment

This module does not have an exam but a series of 'Labs' which are Javascript questions which you have to program. The last assignment is a validation project worth 40%

Insiders Hints and Tips

Depending on what educational background you come from this subject could be the most difficult of the first semester. It requires learning the basics of coding languages and it can be very difficult to grasp at first. You will be consistently challenged with each Lab Practical getting increasingly more difficult but you will be surprised with your ability and how much you have learned by the end of it. Personally this took a long time to click with me but if you take a systematic approach, ask for learning support if needed and make sure to ask the help of your lecturer you will grasp it.
